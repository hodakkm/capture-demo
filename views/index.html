OCR Application

Test application for Cloud Capture

User Stories

  1. I can submit a FormData object that includes a file upload to the /process route.
  2. When I submit something, I will receive a taskID in the JSON response.
  3. I can submit the taskID to the /task route and receive the JSON response
  4. When the JSON response is returned from /task, it should be parsed to only show the annotations
